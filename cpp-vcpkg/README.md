## Installation
    $ git clone -b typescript-react https://github.com/LiuToki/project-templates.git

or

    $ wget https://github.com/LiuToki/project-templates/archive/refs/heads/cpp-vcpkg.zip
    $ unzip cpp-vcpkg.zip

## Features
- C++
- CMake
- vcpkg

## Initialize
```
$ git submodule init
```

## Build
```
$ mkdir build
$ cd build
$ cmake -G Ninja ..
$ ninja
```
